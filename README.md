# AZ-900 Study Guide (Azure Fundamentals)

> A practical roadmap to pass the Microsoft Azure Fundamentals (AZ-900) exam — based on my 1-day preparation strategy.

---

## About This Repo

I prepared for the AZ-900 exam in a very short time using:

- Practice questions (~200)
- A quick cram/tutorial video
- Pattern recognition instead of deep memorization

This repo is meant to help anyone who wants to **pass efficiently** without getting overwhelmed.

---

## How to Study for AZ-900

### Step 1: Understand the Exam Style
- AZ-900 is **concept-based**, not deeply technical
- Focus on:
  - Recognizing services
  - Understanding use cases
  - Choosing the *best Azure solution*

---

### Step 2: Practice Questions First
- Do **100–200 practice questions**
- Focus on:
  - Why the correct answer is right
  - Why others are wrong

This builds **pattern recognition**, which is key for AZ-900.

---

### Step 3: Watch a Cram Video
- Watch a **2–4 hour tutorial (1.25x–1.5x speed)**
- Focus on:
  - Service models (IaaS, PaaS, SaaS)
  - Core Azure services
  - Governance tools
  - Networking basics

---

### Step 4: Focus on High-Yield Concepts

#### Service Models
- IaaS → You manage OS
- PaaS → Azure manages OS
- SaaS → You just use software

#### Governance
- RBAC → Who can access
- Policy → What is allowed
- Locks → Prevent deletion

#### Architecture
- Region → geographic location
- Availability Zone → datacenter redundancy
- Resource Group → container for resources

#### Monitoring
- Azure Monitor → logs & metrics
- Azure Advisor → recommendations
- Service Health → Azure outages

#### Identity
- Microsoft Entra ID → identity management
- MFA → extra security
- SSO → single login

---

### Step 5: Final Revision Strategy
- Review weak areas
- Do 20–30 more questions
- Focus on **confusing topics only**

---

## Exam Strategy

- Don’t overthink simple questions  
- Eliminate 2 wrong answers  
- Choose the most **Azure-native / managed solution**  

If stuck:
> Answer → Flag → Move → Come back later

---

## Key Tips

- AZ-900 is about **recognition, not memorization**
- Focus on **understanding use cases**
- Practice questions matter more than videos
- Stay calm — you only need ~65–70% to pass
